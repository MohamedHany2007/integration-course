المتطابقات المثلثية الهامة

متطابقات أساسية

\(\sin^2 x + \cos^2 x = 1 \quad|\quad 1+\tan^2 x = \sec^2 x \quad|\quad 1+\cot^2 x = \csc^2 x\)

\(\sin^2 x = \dfrac{1-\cos 2x}{2} \quad|\quad \cos^2 x = \dfrac{1+\cos 2x}{2}\)

\(\sin 2x = 2\sin x\cos x \quad|\quad \cos 2x = \cos^2 x - \sin^2 x\)

\(\tan^2 x = \sec^2 x - 1 \quad|\quad \cot^2 x = \csc^2 x - 1\)

المتطابقات الزائدية الهامة

دوال زائدية

\(\cosh^2 x - \sinh^2 x = 1 \quad|\quad 1 - \tanh^2 x = \text{sech}^2 x \quad|\quad \coth^2 x - 1 = \text{csch}^2 x\)

\(\sinh x = \dfrac{e^x - e^{-x}}{2} \quad|\quad \cosh x = \dfrac{e^x + e^{-x}}{2} \quad|\quad \tanh x = \dfrac{e^x - e^{-x}}{e^x + e^{-x}}\)

صيغ أويلر

\(e^{ix} = \cos x + i\sin x \quad|\quad \cos x = \dfrac{e^{ix}+e^{-ix}}{2} \quad|\quad \sin x = \dfrac{e^{ix}-e^{-ix}}{2i}\)

التكامل غير المحدود
تعريف

إذا كانت \(g\) دالة معرفة على فترة ما، وكانت \(F\) دالة مشتقة على هذه الفترة بحيث يكون \(F'(x) = g(x)\)، فإن الدالة \(F\) تُسمى دالة مقابلة أو تكاملاً غير محدود للدالة \(g\)، ونعبر عن ذلك بالرمز:

\[\int g(x)\,dx = F(x) + c\]

حيث \(c\) ثابت اختياري يُسمى ثابت التكامل.

الجدول القياسي للتكاملات

مالصيغة
١\(\displaystyle\int 0\,dx = c\)
٢\(\displaystyle\int k\,f(x)\,dx = k\int f(x)\,dx\)  — المقدار الثابت يخرج من التكامل
٣\(\displaystyle\int [f(x)+g(x)]\,dx = \int f(x)\,dx + \int g(x)\,dx\)
٤\(\displaystyle\int x^n\,dx = \dfrac{x^{n+1}}{n+1}+c \quad (n\neq -1)\)
٥\(\displaystyle\int \frac{1}{x}\,dx = \ln|x|+c\)
٦\(\displaystyle\int e^x\,dx = e^x+c\)
٧\(\displaystyle\int a^x\,dx = \dfrac{a^x}{\ln a}+c\)
٨\(\displaystyle\int \sin x\,dx = -\cos x+c\)
٩\(\displaystyle\int \cos x\,dx = \sin x+c\)
١٠\(\displaystyle\int \sec^2 x\,dx = \tan x+c\)
١١\(\displaystyle\int \csc^2 x\,dx = -\cot x+c\)
١٢\(\displaystyle\int \sec x\tan x\,dx = \sec x+c\)
١٣\(\displaystyle\int \csc x\cot x\,dx = -\csc x+c\)
١٤\(\displaystyle\int \dfrac{dx}{\sqrt{1-x^2}} = \sin^{-1}x+c\)
١٥\(\displaystyle\int \dfrac{dx}{1+x^2} = \tan^{-1}x+c\)
١٦\(\displaystyle\int \dfrac{dx}{x\sqrt{x^2-1}} = \sec^{-1}x+c\)
١٧إذا كان البسط = مشتقة المقام: \(\displaystyle\int \dfrac{f'(x)}{f(x)}\,dx = \ln|f(x)|+c\)

المعادلة التفاضلية

تعريف

المعادلة التفاضلية في متغيرين \(x,y\) هي معادلة تحتوي على المتغيرين وتفاضلات \(y\) بالنسبة إلى \(x\). المعادلة التفاضلية من الدرجة الأولى والرتبة الأولى تكون على الصورة:

\[f\!\left(x,y,\frac{dy}{dx}\right)=0\]

لحل المعادلة التفاضلية نضرب طرفيها ثم نكامل كلاً منهما على حدة.

أمثلة محلولة

مثال (١) — تكامل مباشر لكثيرة حدود▼
احسب: \(\displaystyle\int (2x^3 - 4x^2 + x)\,dx\)
١
نوزع التكامل على كل حد:
\[\int 2x^3\,dx - \int 4x^2\,dx + \int x\,dx\]
٢
نطبق قانون \(\int x^n dx = \dfrac{x^{n+1}}{n+1}\) على كل حد:
\[\dfrac{x^4}{2} - \dfrac{4x^3}{3} + \dfrac{x^2}{2} + c\]
مثال (٢) — تكامل يحتوي كسوراً وجذوراً▼
احسب: \(\displaystyle\int\!\left(8x^3 - 6x^{\frac{1}{2}} + \dfrac{1}{x^2}\right)dx\)
١
نحوّل \(\dfrac{1}{x^2} = x^{-2}\) ونطبق القانون على كل حد:
\[8\cdot\dfrac{x^4}{4} - 6\cdot\dfrac{x^{3/2}}{3/2} + \dfrac{x^{-1}}{-1} + c\]
\[2x^4 - 4x^{3/2} - \dfrac{1}{x} + c\]
مثال (٣) — معادلة تفاضلية (١)▼
أوجد الحل العام للمعادلة التفاضلية: \(\dfrac{dy}{dx} = 5\)
١
نضرب الطرفين في \(dx\):
\[dy = 5\,dx\]
٢
نكامل الطرفين:
\[\int dy = \int 5\,dx\]
\[y = 5x + c\]
مثال (٤) — معادلة تفاضلية (٢)▼
أوجد الحل العام للمعادلة التفاضلية: \(\dfrac{dy}{dx} = x^2 + 1\)
١
نفصل المتغيرات:
\[dy = (x^2+1)\,dx\]
٢
نكامل الطرفين:
\[y = \dfrac{x^3}{3} + x + c\]

أمثلة محلولة

مثال (١) — تطبيق مباشر▼
احسب: \(\displaystyle\int \dfrac{\sin x}{\cos^2 x}\,dx\)
١
نعيد كتابة الكسر:
\[\int\frac{\sin x}{\cos x}\cdot\frac{1}{\cos x}\,dx = \int\tan x\cdot\sec x\,dx\]
\[\sec x + c\]
مثال (٢) — استخدام قاعدة \(\int u^n du\)▼
احسب: \(\displaystyle\int\sin^3 x\cos x\,dx\)
١
نلاحظ أن \(d(\sin x) = \cos x\,dx\)
\[\int\sin^3 x\,d(\sin x)\]
٢
نطبق قانون \(\int u^n du = \dfrac{u^{n+1}}{n+1}\):
\[\dfrac{\sin^4 x}{4}+c\]
مثال (٣) — باستخدام متطابقة التضاعف▼
احسب: \(\displaystyle\int\sin^2 x\,dx\)
١
نستخدم: \(\sin^2 x = \dfrac{1-\cos 2x}{2}\)
\[\int\frac{1-\cos 2x}{2}\,dx = \frac{1}{2}\int dx - \frac{1}{2}\int\cos 2x\,dx\]
\[\dfrac{x}{2} - \dfrac{\sin 2x}{4}+c\]
مثال (٤) — \(\int\tan x\,dx\) بطريقة اللوغاريتم▼
أثبت أن: \(\displaystyle\int\tan x\,dx = \ln|\sec x|+c\)
١
نكتب \(\tan x = \dfrac{\sin x}{\cos x}\):
\[\int\frac{\sin x}{\cos x}\,dx = -\int\frac{-\sin x}{\cos x}\,dx = -\int\frac{d(\cos x)}{\cos x}\]
٢
نطبق \(\int\dfrac{f'}{f}dx = \ln|f|\):
\[-\ln|\cos x|+c = \ln|\sec x|+c\]

الباب الرابع
التكامل بالتجزيء
قانون التكامل بالتجزيء

من مقرر التفاضل نعلم أن: \(d(u\cdot v) = u\,dv + v\,du\)

بتكامل الطرفين:

\[\boxed{\int u\,dv = uv - \int v\,du}\]
قاعدة LIATE لاختيار u

Logarithmic — Inverse trig — Algebraic — Trigonometric — Exponential
اختر \(u\) من الفئة الأعلى في هذا الترتيب.

أمثلة محلولة

مثال (١) — \(\int xe^x\,dx\)▼
احسب بالتجزيء: \(I = \displaystyle\int xe^x\,dx\)
١
نختار: \(u=x \Rightarrow du=dx\) ، \(dv=e^x dx \Rightarrow v=e^x\)
\[\int xe^x\,dx = xe^x - \int e^x\,dx\]
\[e^x(x-1)+c\]
مثال (٢) — \(\int e^x\sin x\,dx\) (تجزيء مرتين)▼
احسب: \(\displaystyle I = \int e^x\sin x\,dx\)
١
نختار: \(u=\sin x,\; dv=e^x dx\)
\[I = e^x\sin x - \int e^x\cos x\,dx\]
٢
نجزئ مرة أخرى: \(u=\cos x,\; dv=e^x dx\)
\[I = e^x\sin x - e^x\cos x - \int e^x\sin x\,dx = e^x\sin x - e^x\cos x - I\]
٣
نجمع \(I\) في الطرف الأيسر:
\[I = \dfrac{e^x(\sin x - \cos x)}{2}+c\]
مثال (٣) — \(\int\ln x\,dx\)▼
احسب: \(\displaystyle\int\ln x\,dx\)
١
نختار: \(u=\ln x \Rightarrow du=\tfrac{1}{x}dx\) ، \(dv=dx \Rightarrow v=x\)
\[\int\ln x\,dx = x\ln x - \int dx\]
\[x\ln x - x + c\]
مثال (٤) — \(\int\tan^{-1}x\,dx\)▼
احسب: \(\displaystyle\int\tan^{-1}x\,dx\)
١
نختار: \(u=\tan^{-1}x \Rightarrow du=\dfrac{dx}{1+x^2}\) ، \(dv=dx \Rightarrow v=x\)
\[\int\tan^{-1}x\,dx = x\tan^{-1}x - \int\frac{x}{1+x^2}\,dx\]
\[x\tan^{-1}x - \dfrac{1}{2}\ln(1+x^2)+c\]

الباب الخامس
التكامل بالتعويض
الفكرة

نستبدل المتغير \(x\) بتعبير جديد \(y=g(x)\) لتبسيط صورة التكامل. هناك نوعان: تعويضات جبرية (للتخلص من الجذور) وتعويضات مثلثية (للتكاملات ذات الجذور من درجة ثانية).

أولاً: التعويضات الجبرية

مثال (١) — إزالة الجذر التربيعي▼
احسب: \(\displaystyle\int\dfrac{dx}{(3-x)\sqrt{x-2}}\)
١
نعوِّض: \(y=\sqrt{x-2}\) أي \(y^2=x-2\)، ومنها \(2y\,dy=dx\) و \(x=y^2+2\)
\[\int\frac{2y\,dy}{(3-y^2-2)\cdot y} = 2\int\frac{dy}{1-y^2}\]
\[2\tan^{-1}\!\sqrt{x-2}+c\]
مثال (٢) — تعويض لتبسيط الأُس▼
احسب: \(\displaystyle\int x\sqrt{1-x^3}\,dx\) (الصيغة العامة)
١
نعوِّض: \(y=\sqrt{1-x^3}\) أي \(y^2=1-x^3\)، ومنها \(2y\,dy = -3x^2\,dx\)
٢
نعبّر عن التكامل بدلالة \(y\) ونحل:
\[-\dfrac{2}{3}\cdot\dfrac{1}{3}(1-x^3)^{3/2}+c\]

ثانياً: التعويضات المثلثية

الصورةالتعويضالمتطابقة
\(\sqrt{a^2-x^2}\)\(x=a\sin\theta\)\(a^2-a^2\sin^2\theta = a^2\cos^2\theta\)
\(\sqrt{a^2+x^2}\)\(x=a\tan\theta\)\(a^2+a^2\tan^2\theta = a^2\sec^2\theta\)
\(\sqrt{x^2-a^2}\)\(x=a\sec\theta\)\(a^2\sec^2\theta-a^2 = a^2\tan^2\theta\)
مثال (٣) — تعويض مثلثي \(\sqrt{a^2-x^2}\)▼
احسب: \(\displaystyle\int\dfrac{dx}{\sqrt{a^2-x^2}}\)
١
نعوِّض: \(x=a\sin\theta \Rightarrow dx=a\cos\theta\,d\theta\)
\[\int\frac{a\cos\theta\,d\theta}{a\cos\theta} = \int d\theta = \theta + c\]
٢
نرجع للمتغير الأصلي: \(\theta=\sin^{-1}\!\dfrac{x}{a}\)
\[\sin^{-1}\!\dfrac{x}{a}+c\]

الباب السادس
التكامل باستخدام الكسور الجزئية
شرط تطبيق الكسور الجزئية

يجب أن يكون الكسر حقيقياً أي درجة البسط أقل من درجة المقام. إن لم يكن كذلك نقسم أولاً ثم نطبق الكسور الجزئية.

الحالة الأولى — عوامل مختلفة من الدرجة الأولى

إذا كان \(Q(x)=(x-a_1)(x-a_2)\cdots(x-a_n)\):

\[\frac{P(x)}{Q(x)} = \frac{A_1}{x-a_1}+\frac{A_2}{x-a_2}+\cdots+\frac{A_n}{x-a_n}\]
الحالة الثانية — عوامل مكررة من الدرجة الأولى

إذا كان \(Q(x)=(x-a)^r\):

\[\frac{P(x)}{(x-a)^r} = \frac{A_1}{x-a}+\frac{A_2}{(x-a)^2}+\cdots+\frac{A_r}{(x-a)^r}\]
الحالة الثالثة — عوامل تربيعية غير قابلة للتحليل

لكل عامل تربيعي \(ax^2+bx+c\) (لا جذور حقيقية) نخصص كسراً من الشكل:

\[\frac{Ax+B}{ax^2+bx+c}\]

مثال محلول

مثال — الحالة الأولى▼
احسب: \(\displaystyle\int\dfrac{3x+1}{(x-1)(x+2)}\,dx\)
١
نكتب: \(\dfrac{3x+1}{(x-1)(x+2)}=\dfrac{A}{x-1}+\dfrac{B}{x+2}\)
٢
نوحد المقامات: \(3x+1 = A(x+2)+B(x-1)\)
٣
نضع \(x=1\): \(A=\dfrac{4}{3}\) ، ونضع \(x=-2\): \(B=\dfrac{5}{3}\)
\[\dfrac{4}{3}\ln|x-1|+\dfrac{5}{3}\ln|x+2|+c\]

الباب السابع
التكامل باستخدام إكمال المربع
متى نستخدمه؟

عندما يكون التكامل من الصور: \(\displaystyle\int\dfrac{dx}{ax^2+bx+c}\) أو \(\displaystyle\int\dfrac{dx}{\sqrt{ax^2+bx+c}}\) أو \(\displaystyle\int\sqrt{ax^2+bx+c}\,dx\)

خطوات إكمال المربع

الخطوات التفصيلية

\(ax^2+bx+c = a\!\left[\left(x+\dfrac{b}{2a}\right)^2 - \left(\dfrac{b}{2a}\right)^2 + \dfrac{c}{a}\right] = a\left[(y)^2 \pm A^2\right]\)

حيث \(y = x + \dfrac{b}{2a}\) ، \(A^2 = \left|\dfrac{c}{a}-\left(\dfrac{b}{2a}\right)^2\right|\)

الصيغ القياسية الناتجة

الصيغة بعد الإكمالالنتيجة
\(\displaystyle\int\dfrac{dy}{\sqrt{y^2+A^2}}\)\(\sinh^{-1}\!\dfrac{y}{A}+c\)
\(\displaystyle\int\dfrac{dy}{\sqrt{y^2-A^2}}\)\(\cosh^{-1}\!\dfrac{y}{A}+c\)
\(\displaystyle\int\dfrac{dy}{\sqrt{A^2-y^2}}\)\(\sin^{-1}\!\dfrac{y}{A}+c\)
\(\displaystyle\int\dfrac{dy}{y^2+A^2}\)\(\dfrac{1}{A}\tan^{-1}\!\dfrac{y}{A}+c\)
مثال — إكمال المربع ثم التكامل▼
احسب: \(\displaystyle\int\dfrac{dx}{x^2+4x+13}\)
١
نكمل المربع: \(x^2+4x+13 = (x+2)^2+9\)
٢
نعوِّض \(y=x+2\):
\[\int\frac{dy}{y^2+9} = \frac{1}{3}\tan^{-1}\!\frac{y}{3}+c\]
\[\dfrac{1}{3}\tan^{-1}\!\dfrac{x+2}{3}+c\]

الباب الثامن
التكامل باستخدام الاختزال المتتالي
تعريف

عند محاولة إجراء تكامل بالتجزيء نجد أحياناً أن النتيجة تعتمد على تكامل آخر من نفس الصورة لكن أقل درجة. نستخدم قانون الاختزال المتتالي (Reduction Formula) لإيجاد علاقة بين التكامل \(I_n\) وتكامل أبسط \(I_{n-1}\) أو \(I_{n-2}\).

أشكال قانون الاختزال

الأشكال العامة

• \(I_n = \int f(x,n)\,dx\) يرتبط بـ \(I_{n-1}\) أو \(I_{n-2}\)

• \(I_{n,m} = \int f(x,n,m)\,dx\) يرتبط بـ \(I_{n-2,m}\) أو \(I_{n,m-2}\)

أمثلة محلولة

مثال (١) — قانون الاختزال لـ \(\int\sin^n x\,dx\)▼
اشتق قانون الاختزال لـ \(I_n = \displaystyle\int\sin^n x\,dx\)
١
نكتب \(\sin^n x = \sin^{n-1}x\cdot\sin x\) ونجزئ: \(u=\sin^{n-1}x,\; dv=\sin x\,dx\)
\[I_n = -\sin^{n-1}x\cos x + (n-1)\int\sin^{n-2}x\cos^2 x\,dx\]
٢
نستبدل \(\cos^2 x = 1-\sin^2 x\) ونبسط وننقل \(I_n\):
\[I_n = -\dfrac{\sin^{n-1}x\cos x}{n}+\dfrac{n-1}{n}I_{n-2}\]
مثال (٢) — قانون الاختزال لـ \(\int\sec^n x\,dx\)▼
اشتق قانون الاختزال لـ \(I_n = \displaystyle\int\sec^n x\,dx\)
١
نكتب \(\sec^n x = \sec^{n-2}x\cdot\sec^2 x\) ونجزئ
٢
نطبق التجزيء ونستخدم \(\tan^2 x = \sec^2 x - 1\):
\[I_n = \dfrac{\sec^{n-2}x\tan x}{n-1}+\dfrac{n-2}{n-1}I_{n-2}\]

الباب التاسع
التكامل المحدود (المُعيَّن)
تعريف التكامل المحدود — قاعدة باروو

إذا كان \(F'(x)=f(x)\) وكانت \(f\) دالة متصلة على الفترة المغلقة \([a,b]\)، فإن:

\[\int_a^b f(x)\,dx = \Big[F(x)\Big]_a^b = F(b)-F(a)\]

خواص التكامل المحدود

النظريات الأساسية الخمس

نظرية (١): \(\displaystyle\int_a^b c\,dx = c(b-a)\)  — تكامل ثابت

نظرية (٢): \(\displaystyle\int_a^b cf(x)\,dx = c\int_a^b f(x)\,dx\)  — إخراج الثابت

نظرية (٣): \(\displaystyle\int_a^b [f\pm g]\,dx = \int_a^b f\,dx \pm \int_a^b g\,dx\)  — الخطية

نظرية (٤): إذا \(f(x)\geq 0\) على \([a,b]\) فإن \(\displaystyle\int_a^b f\,dx\geq 0\)  — الإشارة

نظرية (٥): \(\displaystyle\int_a^b f\,dx = \int_a^c f\,dx + \int_c^b f\,dx\quad (a < c < b)\)  — التجزئة

أمثلة محلولة

مثال (١) — تكامل مباشر▼
احسب: \(\displaystyle\int_1^3 x^3\,dx\)
\[\left[\frac{x^4}{4}\right]_1^3 = \frac{81}{4} - \frac{1}{4} = \frac{80}{4}\]
\[= 20\]
مثال (٢) — تكامل مثلثي محدود▼
احسب: \(\displaystyle\int_0^{\pi/2}(3\cos 4x + 7)\,dx\)
\[\left[\frac{3}{4}\sin 4x + 7x\right]_0^{\pi/2}\]
\[= \left(\frac{3}{4}\sin 2\pi + \frac{7\pi}{2}\right) - (0+0) = \frac{7\pi}{2}\]
\[\dfrac{7\pi}{2}\]
مثال (٣) — تكامل محدود بالتجزيء▼
احسب: \(\displaystyle\int_0^1 xe^{2x}\,dx\)
١
نجد التكامل اللامحدود بالتجزيء: \(u=x,\; dv=e^{2x}dx\)
\[\int xe^{2x}dx = \frac{xe^{2x}}{2} - \frac{e^{2x}}{4}+c\]
٢
نطبق الحدود:
\[\left[\frac{xe^{2x}}{2}-\frac{e^{2x}}{4}\right]_0^1 = \left(\frac{e^2}{2}-\frac{e^2}{4}\right)-\left(0-\frac{1}{4}\right)\]
\[\dfrac{e^2+1}{4}\]
